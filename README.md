# PRABHU TAILORS Website

A professional, modern, mobile-friendly landing page created for **PRABHU TAILORS (Ladies & Gents)**, a local tailoring and readymade clothing store in Visakhapatnam.

## 🌟 Features

- **Responsive Design**: Mobile-first styling that looks perfect on smartphones, tablets, and desktops.
- **Modern UI/UX**: Includes glassmorphism effects, smooth scrolling, and scroll-triggered animations.
- **Sections Included**: 
  - Home (Hero Banner)
  - Why Choose Us
  - Custom Tailoring Services
  - Premium Readymade Products
  - Customer Reviews
  - About Us
  - Contact (with integrated Google Map)
- **Quick Contact Options**: Floating Call and WhatsApp action buttons for immediate communication.

## 🛠️ Technologies Used

- **HTML5**: Semantic tags for solid structure and SEO friendliness.
- **CSS3 (Vanilla)**: Custom styling without heavy frameworks, utilizing CSS Grid and Flexbox.
- **JavaScript (Vanilla)**: Lightweight scripts for mobile menu toggling and scroll observation animations.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: Uses the 'Outfit' font family for a premium look.

## 🚀 How to Run

Because this project relies entirely on basic web technologies (HTML/CSS/JS), there is no build process or server required for local viewing.

1. Clone or download this repository.
2. Navigate to the project directory.
3. Open `index.html` in any modern web browser (Chrome, Safari, Edge, Firefox).
